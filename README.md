# Dugsiiye Skills

AI Skills built for Somalis.

A collection of AI Skills designed around the tools, services, and contexts Somalis actually use — from local payment systems like EVC Plus and Zaad, to Somali-language workflows, to anything else that's missing from the default toolbox of today's AI assistants.

Skills here are written to work with Claude, ChatGPT, and other AI systems that support the Skills format.

## What's a Skill?

A Skill is a folder containing a `SKILL.md` file that teaches an AI assistant how to do a specific task well. The AI loads the skill only when it's relevant, so each skill stays focused and lightweight.

## Planned

- Somali payment integrations (EVC Plus, Zaad, Sahal, etc.)
- Somali-language writing, translation, and formatting
- Local business and document workflows
- More as the community contributes
